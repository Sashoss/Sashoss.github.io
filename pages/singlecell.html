Ideally we would look at each cluster markers predicted in the previous step and assign a unique cell type label. Here, lets lets label each cell using its gene expression patterns.

library(SingleR)
library(celldex)

# Prepare a SingleR reference dataset, or use one of the built-in references
# Human Primary Cell Atlas (HPCA) reference dataset for human
hpca.se <- HumanPrimaryCellAtlasData()

# Extracting expression data from Seurat object for SingleR
expression_data <- GetAssayData(seurat_object, assay = "RNA", slot = "data")

# Run SingleR
singleR_results <- SingleR(test = expression_data, ref = hpca.se, labels = hpca.se$label.main)

Lets plot cell types predicted by SingleR. Cell labels shown below look mostly consistent with the labels added at Broad Institute Single cell database entry page, although Astrocytes and Neurons are not expected cell types in renal cell carcinoma single cell data. These cell are likely mislabeled, and requires manual investigation by looking at its gene markers predicted in the clustering step, and effective communication with the bench scientists.

# Add SingleR predictions to the Seurat object metadata
seurat_object$predicted_cell_type <- singleR_results$labels

# Plot UMAP with cells colored by their predicted cell types
DimPlot(seurat_object, reduction = "umap", group.by = "predicted_cell_type")
